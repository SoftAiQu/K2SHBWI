### "What is K2SHBWI?"
K2SHBWI is a format that combines images with interactive features in a single, offline-capable file. Like an image, but clickable!

**Full answer:** See `01-general-questions.md`

### "How do I create a K2SH file?"
```bash
pip install k2shbwi
k2shbwi create image.png output.k2sh
k2shbwi view output.k2sh --web
```

**Full answer:** See `02-usage-questions.md`

## âš¡ Most Common Questions

### Q: Is K2SHBWI free?
**A:** Yes, completely free and open-source (MIT License)

### Q: Can I use K2SHBWI commercially?
**A:** Yes, MIT License allows commercial use

### Q: Is K2SHBWI secure?
**A:** Yes, encryption support coming Q4 2026

### Q: Can I edit K2SH files?
**A:** Currently create or view. Editing coming in future version

### Q: What file types can I convert?
**A:** HTML, PDF, PowerPoint (PPTX), and standard images

### Q: How large can K2SH files be?
**A:** No hard limit, but tested up to 5GB with compression

### Q: Do I need internet to use K2SH files?
**A:** No, 100% offline support

### Q: Can I share K2SH files?
**A:** Yes, just like any file (email, cloud, USB, etc.)
